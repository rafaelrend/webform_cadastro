Exemplo de cadastro usando webforms, .net versão 2.0

Acesso a dados com Ado.net para permitir utilizar em servidores que não possuem Full Trust (Rendlibrary)

Mesmo cadastro do projeto neste link:
https://github.com/rafaelrend/mvc_cadastro

A diferença é que este projeto utiliza webforms, e, por ser um modelo mais antigo, quando fora feito ainda não existia bootstrap. O Layout CSS
é próprio, criado por volta de 2009.

Este projeto foi colocado no github para servir de referência, quando houver a necessidade de manutenção em sistemas antigos.
